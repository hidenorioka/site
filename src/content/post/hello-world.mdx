# はじめに

このサイトにブログ機能を追加しました！

## 特徴

- **MDX対応**: Markdownに加えて、Reactコンポーネントも使用可能
- **タグシステム**: 記事をカテゴリ別に整理
- **日付順ソート**: 最新の記事が上部に表示

## MDXの例

MDXを使用することで、通常のMarkdownに加えて以下のような機能も利用できます：

```jsx
const greeting = "Hello, World!";
console.log(greeting);
```

今後は技術的な内容や日常の出来事など、様々な記事を投稿していく予定です。
